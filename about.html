At USF I have continued to learn more C, C++ and Python, while in summer 2025 I took the Global Career Accelerator where I learned to work with Python and SQL to analyze databases, while also studying HTML, CSS and JavaScript independently to begin developing this website. In December 2025 and 2026 I continued my full-stack learning by completing freeCodeCamp's JavaScript Development certification, projects from these mentioned experiences can be found here.
